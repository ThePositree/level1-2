# TodoApp

***
## Приложение:
Это список дел для того чтобы контролировать и не забывать дела, которые нужно сделать за день.
***
## Стек:
  - React.js
  - SCSS
***
## Скрипты:
1. Для запуска в режиме разработки:
```
npm run dev
```
2. Для сборки:
```
npm run build
```
