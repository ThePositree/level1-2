# TodoApp

***
## Приложение:
Это список дел для того чтобы контролировать и не забывать дела, которые нужно сделать за день.
***
## Стек:
  - React.js
  - Redux/toolkit
  - SCSS
***
## Скрипты:
1. Для установки зависимостей:
```
npm install
```
2. Для запуска в режиме разработки:
```
npm run dev
```
3. Для сборки:
```
npm run build
```
